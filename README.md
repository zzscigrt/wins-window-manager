## Wins 到底是个什么软件

简单说，Wins 是一款专为 macOS 设计的多窗口管理工具，支持苹果 M 系芯片，核心卖点是：**拖窗口到屏幕边缘就能快速分屏**，配合悬浮分屏、Dock 预览、自定义快捷键，基本把原生 Mac 窗口管理的几个短板都补上了。

它的设计理念很克制——没有菜单栏图标，没有独立窗口，设置面板直接集成进系统设置里，就好像原本就是系统功能一样。性能也做得不错，内存占用一般低于 30MB，CPU 使用大多数情况下只有 0.1% 左右。

## 核心功能说说

**悬浮分屏 & 快速分屏**

把窗口拖到屏幕边缘，就能弹出全屏 / 半屏 / 四分之一等布局选项，选好松手，完事。拖到屏幕顶部悬浮区还能触发悬浮分屏。如果嫌鼠标麻烦，也可以自定义快捷键来操作左右半屏、上下半屏、全屏、隐藏全部等动作。

这个体验流畅度很关键——Wins 做了专门的动画效果，触感不输原生。

**Dock 窗口预览**

鼠标悬停在 Dock 图标上，就能预览该应用当前所有打开的窗口。开了七八个 Finder 窗口？不用一个个点了，悬一下就看清楚了。

**Cmd-Tab Plus**

系统原生的 ⌘-Tab 一直有个问题：只能切换应用，看不到具体窗口。Wins 的 Cmd-Tab Plus 把这个体验大幅增强，支持预览、切换、关闭，方向键也能用，号称是软件「有史以来最强大的功能」。

**调度中心 Pro & 其他细节**

在调度中心里直接用 ⌘+W 关窗口、⌘+Q 退出应用；摇晃窗口一键隐藏其他窗口（Aero Shake）；多屏用户可以一键把窗口移到另一台显示器……这些都是日常高频操作，攒起来省的时间挺可观的。
